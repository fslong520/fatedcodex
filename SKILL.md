# 诡世命簿

## 概述

回合制异世界人生模拟游戏。玩家穿越到随机生成的异世界，从出生开始，每回合做出选择，经历一生，见证世界变迁。

**核心玩法**：
- 🌍 随机生成世界观（魔法体系、种族、国家、历史）
- 👤 随机出身（种族、家庭、天赋、命运）
- ⏳ 回合制推进（每回合=1年或特定阶段）
- 🎲 事件选择（随机事件+命运事件+玩家选择）
- 📖 故事生成（自动记录人生故事）
- 🖼️ 图片生成（关键场景、角色卡）
- 📚 终局成书（人生结束时生成完整短篇小说）

## 快速开始

### 开始新游戏

说：**"开始新游戏"** 或 **"新的人生"**

系统会：
1. 生成随机世界观（可指定偏好）
2. 随机玩家出身
3. 进入婴儿阶段第一回合

### 继续游戏

说：**"继续游戏"** 或 **"下一回合"**

系统会：
1. 推进时间
2. 生成该回合事件
3. 提供选择
4. 记录故事

### 存档管理

- **"保存游戏"** — 保存当前进度
- **"读取存档"** — 加载已有存档
- **"查看存档列表"** — 显示所有存档

## 游戏流程

```
新游戏 → 世界观生成 → 出身生成 → 【命名环节】→ 第一回合
    ↓
回合循环：
    显示状态 → 生成事件 → 玩家选择 → 执行结果 → 记录故事 → 下一回合
    ↓
人生结束 → 终局成书 → 可选：重新开始
```

### 命名环节

角色生成完毕后，进入命名环节：

1. 显示随机生成的角色名
2. 询问玩家是否要改名
3. 玩家可以：
   - 接受默认名
   - 输入新名字（限一次）
   - 让系统重新随机生成

**命名输出格式：**

> **📜 命运已为你写下名字...**
>
> 你的名字是：**{随机名}**

| 选项 | 行动 |
|------|------|
| [1] | 接受这个名字 |
| [2] | 我想改名 |
| [3] | 重新随机一个 |

## 事件系统

### 事件类型

1. **随机事件** — 根据概率触发
2. **命运事件** — 预设的人生节点
3. **世界事件** — 全球性大事
4. **关系事件** — 人际关系相关
5. **选择事件** — 玩家主动选择触发

### 事件权重

事件触发考虑：
- 玩家属性（智力高→学习事件多）
- 人生阶段（少年期→教育事件）
- 世界状态（战争时期→征兵事件）
- 关系网络（贵族朋友→社交事件）
- 前置事件（之前的选择影响后续）

## 终局成书

### 触发条件

当角色死亡或人生结束时，自动进入「终局成书」流程，将整个游戏过程生成一部完整的短篇小说。

### 成书流程

```
人生结束 → 统计数据 → 生成章节大纲 → 撰写小说 → 生成封面 → 导出文件
```

### 小说结构

**标准章节结构：**

| 章节 | 内容 |
|------|------|
| 序章 | 世界观背景、出生 |
| 第1章 | 婴儿期（0-3岁） |
| 第2章 | 童年期（4-12岁） |
| 第3章 | 少年期（13-17岁） |
| 第4章 | 青年期（18-35岁） |
| 第5章 | 中年期（36-55岁） |
| 第6章 | 老年期（56岁+） |
| 尾声 | 死亡、遗产、后世评价 |
| 附录 | 属性记录、成就列表、关键选择 |

### 写作风格

根据世界观自动调整：

| 世界类型 | 风格 | 特点 |
|----------|------|------|
| 奇幻中世纪 | 古典优雅 | 诗意描写、史诗感 |
| 东方玄幻 | 空灵飘逸 | 古风语言、意境深远 |
| 蒸汽朋克 | 机械硬朗 | 工业感、齿轮意象 |
| 黑暗奇幻 | 压抑阴郁 | 克苏鲁风、悬疑感 |
| 异能都市 | 现代简洁 | 快节奏、都市感 |
| 废土末世 | 粗砺真实 | 生存感、荒凉 |

### 叙事视角

- **主视角：** 第三人称有限视角（聚焦主角）
- **重要NPC：** 可切换视角补充剧情
- **关键选择：** 第一人称内心独白

### 输出文件

**生成文件：**

```
stories/[角色名]_[世界名]_[日期]/
├── [角色名]的一生.md        # 完整小说（Markdown）
├── [角色名]的一生.txt       # 纯文本版本
├── 封面.png                 # 小说封面（角色终局图）
├── data/
│   ├── world.json           # 世界数据
│   ├── player.json          # 角色数据
│   └── history.json         # 完整事件历史
└── images/
    └── [所有生成的图片]
```

**序章示例：**
